# 📊 BizInsight - AI-Powered Business Dashboard

BizInsight is a modern, responsive business analytics dashboard built using **React**, **Vite**, and **Tailwind CSS**. It features interactive data visualization trends along with an intelligent, automated AI Business Intelligence simulation that provides actionable strategies for store owners with a single click.

---

## 🚀 Live Demo
Experience the live application here: **[https://ai-dashboard-wine-eight.vercel.app]** *(e.g., Vercel or Netlify URL)*

---

## ✨ Features

- **Executive Summary Cards:** Instant snapshot tracking Total Sales, Total Customers, Active Subscriptions, and Refund Requests with positive/negative growth percentages.
- **Visual Performance Analytics:** An interactive and smooth Line Chart built with **Recharts** to track monthly sales trends.
- **Dynamic Inventory & Sales Management:** A robust data table listing inventory status, prices, and units sold, equipped with an instant real-time search filter.
- **AI Business Intelligence Engine:** A specialized smart simulation that mimics an advanced AI model (like Llama 3) to instantly process core metrics and generate bulletproof executive insights.
- **Sleek Modern UI:** Fully responsive design featuring a dark-themed structural sidebar and fluid interactive states using Tailwind CSS classes.

---

## 🛠️ Tech Stack

- **Frontend Framework:** React.js (Vite)
- **Styling & Responsiveness:** Tailwind CSS
- **Data Visualization:** Recharts
- **Icons:** Native Emojis & Icon Typography
